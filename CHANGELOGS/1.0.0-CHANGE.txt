## [V 1.0.0] - 2025-01-01 (Initial Release)

The first official release of the DrewRPG Level System, aiming for the highest level of stability and scalability.
최고 수준의 안정성 및 확장성을 목표로 하는 DrewRPG Level System의 첫 공식 릴리스입니다.

### [Added] / [추가됨]

* **Core Leveling System:** Added player level, experience (EXP), and skill management systems. / 핵심 레벨링 시스템: 플레이어 레벨, 경험치(EXP) 및 스킬 관리 시스템이 추가되었습니다.
* **JSON-Based Configuration:** All skill definitions, EXP acquisition logic, and level rewards are fully customizable via `skills/*.json` and `rewards/*.json`. / JSON 기반 설정: 모든 스킬 정의, 경험치 획득 로직, 레벨 보상이 `skills/*.json` 및 `rewards/*.json`을 통해 완벽하게 커스터마이징 가능합니다.
* **Data Integrity System:** Logic implemented to prevent Data Inconsistency (Desync) through centralized memory caching and asynchronous I/O. / 데이터 무결성 시스템: 중앙 집중식 메모리 캐시 및 비동기 I/O를 통해 데이터 불일치(Desync)를 방지하는 로직이 적용되었습니다.
* **Administrator Commands:** Essential administrator commands such as `/drpg reload`, `/drpg validate`, `/drpg info`, and `/drpg edit` have been added. / 관리자 명령어: `/drpg reload`, `/drpg validate`, `/drpg info`, `/drpg edit` 등 필수 관리자 명령어가 추가되었습니다.
* **PlaceholderAPI Integration:** Allows the linking of level data with other plugins on the server via PlaceholderAPI. / PlaceholderAPI 연동: 플레이스홀더를 통해 서버 내 다른 플러그인과 레벨 데이터를 연동할 수 있습니다.
* **Security and Support Documentation:** Essential documents like `SECURITY.md`, `SUPPORT.md`, and `LICENSE` are included. / 보안 및 지원 문서: `SECURITY.md`, `SUPPORT.md`, `LICENSE` 등의 필수 문서가 포함되었습니다.
