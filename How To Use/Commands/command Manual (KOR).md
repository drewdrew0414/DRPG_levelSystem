## 1. 일반 사용자 명령어
플레이어가 자신의 현재 레벨과 경험치 상태를 확인하는 명령어입니다.
/rpglevel
 * 설명: 명령어를 실행한 플레이어의 모든 기술(Skill) 레벨과 현재 경험치를 채팅창에 출력합니다.
 * 권한: 모든 플레이어 사용 가능
 * 출력 예시:
   [ PlayerName Stat ]
- Farming : Lv.15 (120.5)
- Mining : Lv.10 (50.0)

## 2. 관리자 명령어 개요
관리자용 명령어는 /drpg ls로 시작하며, 서버 운영자(OP) 권한이 필요합니다.
| 명령어 형태 | 설명 |
|---|---|
| /drpg ls <player> | 특정 플레이어의 레벨 정보를 확인합니다. |
| /drpg ls <add|remove|set> | 특정 플레이어의 레벨 또는 경험치를 수정합니다. |
| /drpg ls reset | 플레이어의 특정 기술 또는 전체 데이터를 초기화합니다. |
| /drpg ls reload | 플러그인 설정 및 보상 데이터를 다시 불러옵니다. |

## 3. 데이터 조작 명령어 상세
플레이어의 기술 데이터를 실시간으로 수정할 수 있습니다. 수정한 후에는 자동으로 레벨업 조건 충족 여부를 확인합니다.
레벨 및 경험치 수정
/drpg ls <action> <player> <skill> <type> <value>
 * action: 수행할 작업
   * add: 현재 값에 추가
   * remove: 현재 값에서 차감 (최소 0)
   * set: 특정 값으로 고정
 * player: 대상 플레이어 이름
 * skill: 수정할 기술 명칭 (예: Farming)
 * type: 수정 대상 (level 또는 exp)
 * value: 정수형 수치
데이터 초기화
/drpg ls reset <player> <skill|all>
 * skill: 특정 기술 이름을 입력하면 해당 기술만 0으로 초기화됩니다.
 * all: 플레이어의 모든 기술 레벨과 경험치를 0으로 초기화합니다.
   
## 4. 시스템 관리 명령어
서버를 재시작하지 않고도 변경된 설정 파일(.json)을 즉시 적용할 수 있습니다.
/drpg ls reload <category>
 * all: 모든 기술, 보상, 기본 설정을 재로드합니다.
 * skill: skills 폴더 내의 기술 정의 파일을 재로드합니다.
 * reward: rewards 폴더 내의 보상 설정 파일(예: Farming.json)을 재로드합니다.
 * config: 플러그인의 기본 config.json 파일을 재로드합니다.
